# サウナ施設検索アプリ

Google APIを利用した、サウナ施設を検索することができるウェブアプリケーションです。
お気に入りの施設を保存したり、施設をレビュー、施設情報を編集することができます。

## 概要
このアプリケーションは、Railsを使用して開発したサウナ施設検索アプリです。  
キーワード検索、地域検索、条件検索が可能となっています。   
ログインすることで施設の詳細情報の編集や施設のレビュー、お気に入り登録、  
行きたい施設の登録をすることができます。

## 作成した背景
学生時代から地元のスーパー銭湯に通っており、その施設のサウナにもよく入っていました。  
上京してからもサウナに行くようになりましたが、旅行やお出かけ先で訪れた際に行きたいと考えている施設を  
その度に検索するのが煩わしいと感じたので、その施設を登録しておいたり、気に入った施設を登録して、  
簡単に閲覧でき、施設情報もわかるものがあればいいなと思い作成しました。

## 機能紹介
* トップページ  
・キーワードからサウナ施設を検索できます。  
・地域での検索も可能です。都道府県を選択後、市区町村が選択できるようになっています。  
・下にスクロールするとログインユーザーのレビューを見ることができます。
<img width="600" height="300" alt="トップページ" src="https://github.com/user-attachments/assets/c63eb604-ec7a-4d77-8156-5712ef290263">

* ユーザー認証機能（ログイン・ログアウト）  
・deviseを利用したログイン機能です。

* 条件検索機能
・施設の条件による検索ができます。  
・モーダルウィンドウを採用しており、「条件で探す」を押下すると、  
　検索欄が出てくる仕組みになっています。

* リストページ  
・検索したサウナ施設をリストで表示しています。
<img width="600" height="300" alt="リストページ" src="https://github.com/user-attachments/assets/848af744-d3bf-44d8-a9b1-7cafdc158503">

* 詳細ページ  
・施設のお気に入り登録・削除ができます。  
・行ってみたい施設の登録・削除ができます。  
・施設の営業時間や設備情報などを編集し、登録できます。
<img width="600" height="300" alt="詳細ページ" src="https://github.com/user-attachments/assets/a4acccce-4eb0-4e5d-9238-0a843b5f60e7">

* 施設のレビューの投稿  
・施設のレビューを投稿することができます。  
<img width="600" height="300" alt="レビューページ" src="https://github.com/Ryutaro032/sauna-app/assets/72485351/65920c29-4d30-4a43-bf70-e1da535fe762">

* 投稿にいいねができる  
・トップページに表示されているレビューのハートマークを押下するといいねができます。

## 機能一覧
| 機能名       | 説明                                |
|------------|---------------------------------------|
| ログイン機能 | サインアップ・サイインイン・ログアウト       |
| 施設検索機能   | キーワードなどから施設を検索できます。    |
| 施設詳細登録機能   | 施設の詳細情報を登録できます。         |
| お気に入り機能     | 施設をお気に入りとして保存できます。    |
| 行きたい施設登録機能 | 行きたい施設として保存できます。  　　　|
| 投稿機能      | 施設ごとに施設に対しての投稿ができます。    |
| いいね機能   | 投稿に対していいねをつけることができます。   |
| プロフィール機能 | プロフィール情報を編集できます。　      |
